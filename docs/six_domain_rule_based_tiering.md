# Six-Domain Evidence Annotation and Rule-Based Tiering

Candidate features are annotated across six evidence domains:

- evidence reproducibility;
- direction consistency;
- effect strength;
- disease-process coverage;
- mechanistic relevance;
- model support.

Tier I is interpreted as a rule-based evidence tier requiring both mechanistic mapping and model-derived support, where that remains the final rule. This is not described as a full score-ranking biomarker panel.

Supplementary Tables S4/S5 should be read as evidence annotation, scoring-rule documentation, and tier reconstruction support, not as proof of a clinically validated diagnostic biomarker panel.
