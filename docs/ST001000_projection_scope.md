# ST001000 Projection Scope

ST001000 is used in this repository only as a summary-level external faecal metabolomics projection and recoverability audit for the metabolomics branch. No end-to-end external validation claim is made for the full IBD-EMSA framework.

## Included

- Release-screened scripts, summaries, manifests, and audit outputs where available.
- Summary-level UC vs Control AUROC/AUPRC reporting: 0.892 / 0.935.
- Documentation of feature matching and projection scope where available.

## Not Claimed

- No end-to-end external validation claim is made for the full IBD-EMSA framework.
- No complete row-level ST001000 recomputation claim is made unless a release explicitly includes row-level labels, prediction scores, matched-feature tables, preprocessing/scaling notes, locked coefficient mapping, and ROC/PR source tables.

## Technical Boundary

Cross-platform metabolomics projection is limited by metabolite nomenclature, annotation confidence, ion mode, feature coverage, batch structure, preprocessing and quantitative scaling. This is a general bottleneck for multi-cohort metabolomics translation in IBD; full external validation requires prospectively harmonised paired multi-omics cohorts or fully documented feature-level mapping.
