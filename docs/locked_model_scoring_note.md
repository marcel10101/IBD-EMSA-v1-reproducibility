# Locked Model Scoring Note

This public package includes recovered coefficient tables and a strict scoring interface. Strict scoring is limited to task x modality rows with recovered logistic coefficients, intercept, feature order, and scaler mean/sd. Rows marked as interpretation-proxy or conditional fusion inputs are documentation and should not be used as formal new-sample probabilities.

ST001000 metabolomics remains summary-level/provisional because strict metabolome coefficient-based projection ingredients are not fully recovered.
