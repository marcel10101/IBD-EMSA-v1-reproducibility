The corrected Figure 2E human cohort sample-source panel was generated using the reproducible script `build_Figure2_panel_E_human_sample_source_fixed.py`. The script reads `Supplementary_Table_S3_Evidence_Map_revised_Figure2E_human_sample_fixed.xlsx`, sheet `Figure2E_human_sample_source`, and exports SVG, PDF, 600-dpi TIFF and PNG preview files. Animal, cell, organoid and other mechanistic model components are excluded from the Figure 2E human sample-source counts and are retained only as mechanistic context in the supplementary evidence map.
