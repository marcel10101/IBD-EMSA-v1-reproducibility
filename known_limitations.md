# Known Limitations and Scope

- ST001000 is a summary-level external faecal metabolomics projection unless explicit row-level recomputation outputs are included in a release.
- IBD-EMSA v1 is not a clinical diagnostic, prognostic, relapse-prediction, or treatment-decision tool.
- The 1,470-feature modelling universe and 499 retained candidate set are documented as aggregate and identifier-harmonisation audit layers unless a complete recomputed bridge file is explicitly supplied.
- Candidate features use six-domain evidence annotation and rule-based evidence-tiering; the repository should not be described as a universal complete score-ranking biomarker panel.
- External projection is affected by feature nomenclature, platform, ion mode, detection coverage, preprocessing, and scale differences.
- Raw restricted third-party data are not redistributed.

- HBI, SCCAI and faecal calprotectin associations are exploratory and may be weak because the score captures molecular eco-metabolic state rather than overt symptom severity.
- Clinical-field missingness and uneven availability in HMP2/IBDMDB reduce power for clinical-correlation analyses.
- Metabolomics-versus-microbiome performance differences are interpreted as functional-output versus upstream ecological-background behaviour, not as clinical deployment evidence.
